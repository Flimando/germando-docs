Häufig gestellte Fragen (FAQ)

Antworten auf die meist gestellten Fragen zu Germando und Pelican

🇩🇪 Über Germando

Was ist Germando?

Germando ist ein spezieller deutscher Fork von Pelican, der komplett ins Deutsche übersetzt wurde. Es behält alle technischen Vorteile von Pelican, aber spricht endlich Deutsch! Perfekt für alle, die ihre Server auf Deutsch verwalten möchten.

Ist Germando dasselbe wie Pelican?

Ja und nein. Germando basiert auf Pelican und verwendet denselben Code, aber ist vollständig ins Deutsche übersetzt. Funktional ist es identisch mit Pelican, nur die Sprache und teilweise die Benutzeroberfläche sind angepasst.

Kann ich von Pelican zu Germando wechseln?

Da Germando auf derselben Codebasis wie Pelican aufbaut, sollte ein Wechsel möglich sein. Jedoch empfehlen wir, erst eine Sicherung zu erstellen und in einer Testumgebung zu testen.

🛠️ Installation & Setup

Welche Systemanforderungen hat Germando?

Germando hat dieselben Anforderungen wie Pelican:

  • PHP 8.1 oder höher
  • MySQL 5.7+ oder MariaDB 10.2+
  • Redis
  • Composer
  • Docker (für Wings)
  • Mindestens 1GB RAM (2GB+ empfohlen)

Kann ich Germando auf Shared Hosting installieren?

Nein, Germando erfordert wie Pelican vollständige Serverkontrolle. Shared Hosting unterstützt normalerweise nicht die erforderlichen Funktionen wie Composer, artisan-Befehle oder Docker.

Ist Germando kostenlos?

Ja! Germando ist wie Pelican 100% kostenlos und Open Source. Du kannst es für persönliche und kommerzielle Zwecke nutzen.

Wie viele Server kann ich mit Germando verwalten?

Es gibt keine Begrenzung in der Software selbst. Die Anzahl hängt von deinen Serverressourcen ab. Jeder Game-Server läuft in einem eigenen Docker-Container.

Kann ich eigene Game-Server-Konfigurationen erstellen?

Ja! Du kannst eigene Eggs erstellen oder bestehende anpassen. Die Egg-Konfigurationen sind JSON-basiert und relativ einfach zu verstehen.

🔧 Technische Fragen

Was ist Wings?

Wings ist der Daemon (Hintergrunddienst), der die eigentlichen Game-Server verwaltet. Während das Panel die Weboberfläche bereitstellt, führt Wings die Server in Docker-Containern aus.

Muss ich Docker verstehen, um Germando zu nutzen?

Grundkenntnisse helfen, aber sind nicht zwingend erforderlich. Germando abstrahiert die meisten Docker-Operationen. Du solltest jedoch die Grundlagen verstehen, um Probleme zu lösen.

Kann ich mehrere Wings-Nodes haben?

Ja! Du kannst mehrere Wings-Instanzen auf verschiedenen Servern betreiben und sie alle über ein Panel verwalten. Dies ermöglicht horizontale Skalierung.

Wie sichere ich meine Germando-Installation?

Wichtige Sicherheitsmaßnahmen:

  • SSL/TLS verwenden (HTTPS)
  • Firewall richtig konfigurieren
  • Zwei-Faktor-Authentifizierung aktivieren
  • Regelmäßige Updates durchführen
  • Starke Passwörter verwenden
  • Datenbank-Backups erstellen

🚨 Fehlerbehebung

Das Panel lädt nicht - was soll ich tun?

Prüfe folgende Punkte:

  1. Ist der Webserver (Apache/Nginx) gestartet?
  2. Sind die PHP-Abhängigkeiten installiert?
  3. Sind die Dateirechte korrekt gesetzt?
  4. Prüfe die Webserver-Logs auf Fehler
  5. Ist die .env-Datei korrekt konfiguriert?

Server starten nicht - was ist das Problem?

Häufige Ursachen:

  • Wings ist nicht gestartet oder nicht erreichbar
  • Docker ist nicht installiert oder läuft nicht
  • Ports sind bereits belegt
  • Unzureichende Serverressourcen
  • Falsche Egg-Konfiguration

Wo finde ich die Log-Dateien?

Wichtige Log-Standorte:

  • /var/www/pelican/storage/logs/ - Panel-Logs
  • /var/log/pelican/ - Wings-Logs
  • /var/log/nginx/ oder /var/log/apache2/ - Webserver-Logs
  • docker logs [container_name] - Container-spezifische Logs

📈 Performance & Skalierung

Wie viele Spieler kann ein Server unterstützen?

Das hängt vom Spiel, der Serverkonfiguration und den verfügbaren Ressourcen ab. Germando selbst setzt keine Limits - die Beschränkungen kommen von den Spielen selbst und deiner Hardware.

Kann ich Germando in der Cloud betreiben?

Ja! Germando funktioniert auf allen großen Cloud-Anbietern wie AWS, Google Cloud, Azure, DigitalOcean, Hetzner, etc. Stelle nur sicher, dass Docker unterstützt wird.

Wie optimiere ich die Performance?

Performance-Tipps:

  • Verwende SSD-Speicher
  • Ausreichend RAM zuweisen
  • Redis für Session-Storage nutzen
  • Regelmäßige Wartung durchführen
  • Monitoring einrichten
