Glossar

Was ist ...?

Panel

Webanwendung, die mit Wings kommuniziert und Ihnen ermöglicht, Ihre Server zu steuern.

Wings

Anwendung, die Ihnen sichere Kontrolle über Ihre Server und über Ihr Panel bietet.

Node

Maschine (Computer oder Server), die Wings ausführt, um sich mit dem Panel zu verbinden.

Server

Instanz eines Containers, der von Wings verwaltet wird.

Docker

Plattform, die es Ihnen ermöglicht, Server in isolierten Containern auszuführen.

Docker Image

Datei, die Bibliotheken, Abhängigkeiten, Dateien usw. enthält, die zum Ausführen eines Containers benötigt werden.

Container

Virtualisierte Umgebung, in der Ihre Server laufen.

Egg

Datei, die konfiguriert, wie ein Docker Image ausgeführt wird.

Yolk

Kuratierte Sammlung von (Kern-) Docker Images, die mit Pelican's Eggs verwendet werden können.

API

Application Programming Interface - ermöglicht es externen Anwendungen, programmatisch mit dem Panel zu interagieren.

Allocation

IP-Adresse und Port-Kombination, die einem Server für Netzwerkkonnektivität zugewiesen wird.

Mount

Eine Möglichkeit, Verzeichnisse vom Host-System in Server-Container zu teilen.

Database Host

Externer Datenbankserver, der verwendet werden kann, um Datenbanken für Spielserver zu erstellen.

Schedule

Automatisierte Aufgaben, die auf Servern in bestimmten Intervallen ausgeführt werden können.

Backup

Komprimiertes Archiv von Serverdateien, das später wiederhergestellt werden kann.

Subuser

Zusätzliches Benutzerkonto mit begrenzten Berechtigungen für den Zugriff auf einen bestimmten Server.

Nest

Sammlung verwandter Eggs, die zusammengefasst sind (in Pelican zugunsten von Tags veraltet).

Location

Logische Gruppierung von Knoten am gleichen geografischen oder Netzwerkstandort (in Pelican zugunsten von Tags veraltet).

SFTP

Secure File Transfer Protocol - ermöglicht sichere Dateiverwaltung für Server.

Console

Echtzeit-Interface, das Server-Ausgaben anzeigt und Befehlseingaben ermöglicht.

Environment Variables

Konfigurationswerte, die an Server weitergegeben werden können, um ihr Verhalten anzupassen.

Resource Limits

Beschränkungen für CPU-, Speicher- und Festplattenutzung für einzelne Server.

Startup Command

Der Befehl, der ausgeführt wird, wenn ein Server gestartet wird.

File Manager

Webbasierte Schnittstelle für die Verwaltung von Serverdateien direkt über das Panel.

Wie es funktioniert:

  1. Benutzer greift über einen Webbrowser auf das Panel zu
  2. Panel bietet die Web-Oberfläche und verwaltet Benutzerauthentifizierung, Server-Konfigurationen und Einstellungen
  3. Wings erhält Befehle vom Panel und verwaltet die tatsächlichen Server-Container
  4. Docker Container bieten isolierte Umgebungen, in denen jeder Spielserver unabhängig läuft
