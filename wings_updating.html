Wings Aktualisierung

Die Aktualisierung von Wings ist ein schmerzloser Prozess und sollte (Obacht sollte!) weniger als eine Minute dauern.

Wings Versionsanforderungen

Panel Version Wings Version Unterstützt
1.0.0+ 1.0.0+ ✅
Warnung: Wenn Sie von < 1.0.0-beta9 aktualisieren, funktioniert dies nicht! Wenn Sie den Befehl nicht verwenden möchten und es manuell machen möchten, folgen Sie dem standardmäßigen Aktualisierungsprozess unten. 
sudo wings update
sudo systemctl restart wings

Update herunterladen

Laden Sie zuerst die aktualisierte Wings Binary in /usr/local/bin herunter. Sie müssen Wings kurz stoppen.

Tipp: Laufende Server werden nicht beeinträchtigt. 
sudo systemctl stop wings
sudo curl -L -o /usr/local/bin/wings "https://github.com/pelican-dev/wings/releases/latest/download/wings_linux_$([[ "$(uname -m)" == "x86_64" ]] && echo "amd64" || echo "arm64")"
sudo chmod u+x /usr/local/bin/wings

Neustart

Starten Sie zweitens den Prozess neu.

sudo systemctl restart wings
